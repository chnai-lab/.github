# CHNAI LAB

A small, Cambodia-based product lab building Khmer-first AI. We ship narrow, working tools for underserved sectors — speech, agriculture, and community moderation — rather than broad platforms.

## What We're Building

- **Kaskor ASR** — Khmer speech recognition, fine-tuned on Whisper. Powers voice-first order intake within Chomkar. Still early; actively improving character error rate.
- **Chomkar** — Agritech coordination platform linking smallholder farmers in Kampong Cham to institutional buyers. Runner-up, Turing Hackathon Cycle 10.
- **The AI Layer (CL-00)** — Shared AI/ML infrastructure and evaluation layer underpinning CHNAI LAB initiatives. In early development.
- **Warden (SAT Digital)** — Khmer-first Telegram moderation tool for Cambodian community operators.

## How We Work

Claim discipline first: we describe what's actually shipped, size markets honestly, and validate with real users before scaling. We'd rather ship something small and honest than something impressive-sounding and unverified.
